# BOJ 오프라인 저지

당신의 채점 결과를 예측해드립니다.

## 사용 방법

### 1. 데이터 셋 준비

작성한 소스코드와 같거나 하위 폴더내에 `.in`, `.out`으로 끝나는 채점 데이터 파일을 준비해주세요.

예시:
```
problem
└── source.py # 채점할 소스
  └── data
    ├── boj
    │ └── sample
    │   ├── 1.in  # 데이터 셋 1
    │   └── 1.out # 데이터 셋 1
    │
    ├── 1.in  # 데이터 셋 2
    └── 1.out # 데이터 셋 2
```

### 2. 간채점기 실행

```bash
python -m boj "채점 소스코드"
```

## 요구 사항

* **Python 3.9** (외부 라이브러리는 사용되지 않았음)

* **사용하고자 하는 언어의 컴파일러** (설치된 경로를 `config.json`안에 적절하게 입력해주세요)
